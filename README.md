## Objective
Develop a highly accurate, real-time face recognition system capable of:
- Smart attendance for employees/students
- Identity verification
- Security & access control applications

## Key Features
- Live webcam face detection & recognition
- High-accuracy embedding comparison (cosine similarity)
- Smart attendance logging with timestamps
- Clean & interactive Streamlit dashboard
- Model comparison (ResNet50 vs InceptionV3 vs ArcFace)

## Technologies Used
- **Face Detection**: MTCNN / RetinaFace
- **Backbone Models**: ResNet50, InceptionV3, InsightFace (ArcFace)
- **Framework**: PyTorch / TensorFlow (transfer learning)
- **Face Alignment**: 5-point or 68-point landmarks
- **Similarity Metric**: Cosine Similarity + Euclidean L2
- **Frontend**: Streamlit (real-time web app)

## Datasets
- Labeled Faces in the Wild (LFW)
- VGGFace2
- Custom team dataset (optional – for final demo)
